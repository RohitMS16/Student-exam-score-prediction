Student Exam Score Prediction
Project Overview

This project is a Machine Learning regression project that predicts student exam scores based on various factors such as study hours, attendance, and previous scores. The goal of this project is to analyze student performance data and build a model that can accurately predict exam scores.

Objective

The main objective of this project is to build a regression model to predict student exam scores and identify the key factors that affect student academic performance.

Dataset

The dataset contains student-related information such as:

Study Hours
Attendance
Previous Scores
Assignments
Exam Score (Target Variable)

Data Preprocessing

Model Building
Model Evaluation
Prediction

Linear Regression
